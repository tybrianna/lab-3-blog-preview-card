## Overview

The objective of this project was to replicate the original image by using html, SASS, and bootstrap. I first created the html to include all the lines and images needed for the file. Next, I added bootstrap and inline text to the different sections to format the document in the proper manner. Once everything looked perfect and in place, I then attempted to use SASS to add color to the background and the bottom paragraph.

### What I learned

I learned how to properly use both bootstrap and SASS. Bootstrap in my opinion is easy to use however, it could become time consuming when doing everything from scratch. SASS is also benefitual when needing to make the same change to several different lines. I got more hands on with both to gain a better undderstanding of them and saw them work in real time.
